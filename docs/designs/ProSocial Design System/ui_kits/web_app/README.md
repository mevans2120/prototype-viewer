# ProSocial Web App — UI Kit

Source: `oestech/prosocial-banking-poc` (staging branch) — Next.js 16 + Tailwind v4.

## Components
- `Nav.jsx` — global nav with logo, points pill, avatar
- `TaskCard.jsx` — post card with engagement counts, preview, action buttons
- `Leaderboard.jsx` — ranked list with top-3 medals
- `LandingHero.jsx` — split-layout landing with steps sidebar
- `Button.jsx`, `PointsPill.jsx`, `PlatformIcon.jsx`, `ActionIcons.jsx`, `LogoMark.jsx`

## Screens demonstrated in `index.html`
1. Landing — hero + step list (unauth state)
2. Session — task feed + leaderboard sidebar (the core view)
3. Post detail — single post with comment suggestions

Click the tabs at the top to switch screens.
